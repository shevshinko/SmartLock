# YouWillBeHacked

Control an electric magnetic lock with via Python webserver (Flask) and a Raspberry Pi (GPIO).

**Prerequisites**

*Hardware*
* Raspberry Pi 4 Model B
* Electric solenoid Lock 12V with Close indicator
* DC Jack Female 5.5mm
* 12V Power adapter
* Relay Module 5v Low
* Green LED
* 100 ohm Resistor
* Jumper Wires

*Software*
* VNC Viewer
* Raspberry Pi Imager
